All of our user stories, to date:

    - I want to find my files easily

    - I want organize my files by file extension

    - I want to use UFO on the command line
        - I want the command line commands to be familiar to me

    - I want to use UFO on my Windows machine

    - I want to use UFO on my Mac

    - I want all my files to stay where they are, so that important files are not misplaced

    - I want to organize the contents of just one folder

Stretch Goals:

    - I want to organize my files based on my own criteria

    - I want to organize the contents of a folder and all its sub-folders
